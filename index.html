Getting Started with Visual Studio Code

Visual Studio Code, also commonly referred to as VS Code, is a source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. To download Visual Studio, first open up this link

https://code.visualstudio.com/Download

Then click on which operating system you use for the download. Once downloaded, you will see an overlay of the user interface,and may change the appearance of VS from there. You can also install support for your favorite programming language, change your keyboard shortcuts, easily migrate from other editors using keymap extensions, customize your editor with themes, and explore VS Code features in the Interactive Editor Playground. If you need more help getting started with Visual Studio, you can watch this video for support.

A console screen is a combination of a display monitor and an input device, usually a keyboard and mouse pair, which allows a user to input commands and receive visual output from a computer or computer system. It is used to maintain or monitor the status of a network or computer. To console code, you will use the command console.log () and enter the the value or var you want to console, in the paranthesis. To open the console screen, go onto the the website you want to see the console code, inspect either via right click or ctrl+shift+i, and click on console to the upper right, on the menu that opened. There you will see all consoled codes if done correctly.
